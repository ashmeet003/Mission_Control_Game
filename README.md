# Mission Control 🚀

A 2D endless runner game built with PyGame where the player navigates a rocket through obstacles.

## 📸 Screenshot

![Gameplay](Game_ScreenShot.png)


## 🎮 Features
- Real-time 4-directional player movement
- Procedural obstacle generation with randomized gaps
- Continuous side-scrolling gameplay
- Collision detection system
- Dynamic score tracking
- Game-over screen with final score display

## Technologies
- Python
- PyGame

   cd Mission_Control_Game
   
2. Install dependencies:
